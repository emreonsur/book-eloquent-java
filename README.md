# Eloquent JavaScript – Study Repo

This repository contains my notes and code while studying the book
**"Eloquent JavaScript Fourth Edition by Marijn Haverbeke"**.

## Structure

- [notes.md](notes.md) – Key takeaways, explanations, and active learning notes.
- `code/` – Example code from the book and its [official website](https://eloquentjavascript.net/), along with my own implementations.
    - `code/lab/` – Free experiments and variations inspired by book examples.
